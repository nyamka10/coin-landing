# 🪙 Coin — лендинг

**Простой и лёгкий лендинг для приложения учёта расходов Coin**
для iOS · Android · Веб

Статика (HTML + CSS + немного JS), которая отдаётся через nginx в Docker-контейнере.

</div>

---

## ✨ Особенности

- 🎨 **Дизайн в стиле Apple** — много воздуха, крупная типографика, спокойные акценты
- 📱 **Адаптивность** — отлично выглядит на десктопе и мобильных
- ⚡️ **Лёгкость** — чистая статика, без бэкенда и базы данных
- 🐳 **Docker** — образ на `nginx:alpine` (~25 МБ) с лимитами ресурсов

## 📂 Структура

| Файл | Назначение |
| --- | --- |
| `index.html` | Разметка лендинга |
| `styles.css` | Стили (Apple-style, адаптив) |
| `app.js` | Анимации появления и поведение навигации |
| `nginx.conf` | Конфиг сервера (gzip, кэширование) |
| `Dockerfile` | Сборка образа на `nginx:alpine` |
| `docker-compose.yml` | Запуск с лимитами CPU и памяти |

## 🚀 Запуск

### Docker Compose (рекомендуется)

```bash
docker compose up -d --build
```

Откройте → **http://localhost:8080**

Остановить:

```bash
docker compose down
```

### Docker без Compose

```bash
docker build -t coin-landing .
docker run -d -p 8080:80 --name coin-landing coin-landing
```

### Локально без Docker

Просто откройте `index.html` в браузере — внешних зависимостей нет.

## 🪶 Почему легко для системы

- Базовый образ `nginx:alpine` — минимальный размер
- Чистая статика: нет Node, БД и бэкенда в рантайме
- Лимиты в Compose: **0.25 CPU** и **64 МБ** памяти
- Включены gzip и кэширование статики
